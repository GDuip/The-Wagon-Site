Batched
Win 7+ Basic CMD

Run a batch file to emulate other apps or run automated command sequences.

  1. Create a new text file.
  2. Change the extension from .txt to .bat.
  3. Right click > Edit, and paste your code.
  4. Double click to run.
CMaDmin
Win 7+ Admin / PE

Forcefully create a new administrator account, bypassing restrictions.

  1. Access an elevated prompt via Windows PE (Repair Computer > Troubleshoot > CMD).
  2. Execute the following commands to add a user:
net user username /add
net localgroup administrators username /add

Login with the new account. You now have full control.

DeTasked
Win 7+ Task Mgr

Kill the monitoring process directly from memory.

  1. Open Task Manager (Ctrl+Shift+Esc or Ctrl+Alt+Del).
  2. Locate the monitoring process (e.g., wpcmon.exe for Parental Controls).
  3. Right click > End Task.
  4. Optional: Right click > "Open File Location" and delete the .exe file permanently.

Swapped (Sticky Keys)
Win 7+ Drive Access

The legendary "Utilman/Sticky Keys" exploit. Swaps accessibility tools with a System-Level Command Prompt.


Method 1: Command Line (Recovery Mode)

  1. Boot into Recovery Mode (Advanced Options > CMD).
  2. Identify your system drive (usually C: or D:). Run dir to check.
  3. Execute swap command:
copy C:\Windows\System32\sethc.exe C:\sethc.bak
copy C:\Windows\System32\cmd.exe C:\Windows\System32\sethc.exe

Reboot. At the login screen, press Shift 5 times. A System CMD will open.

Method 2: Manual Swap

Requires read/write access to the drive (e.g., connecting the drive to another PC).

  1. Navigate to Windows\System32.
  2. Rename utilman.exe to utilman.bak.
  3. Copy cmd.exe and rename the copy to utilman.exe.
  4. Boot back up and click the "Ease of Access" icon at login.
